## 特色
- 极速接入：一分钟**APP**接入设备
- 新分享模式：小程序/H5/独立APP+单个设备绑定，快速分享给好友
- aPaaS低代码生产：常用开发板硬件**代码生成**功能，让硬件开发成为填空题
- 高度自由的开放能力：提供HTTP API，MQTT及Websocket
- 云网关：**同一界面下**统一调度多个设备进行**IO控制**
- 场景：**设备间**通讯，打造个人物联网生态
- 物联规则：突破物联的最后一层障碍
- 物联网新思维：**命令字段** | **通信字段** | **内容字段**
- 提供**开源社区**进行代码交流
